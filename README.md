# Alist-Manager
呃呃呃，用AI做的Alist管理器，目前感觉还行
支持管理多个版本
快捷切换版本
这个批处理文件**需要管理员权限**，配置好之后运行Alist会生成两个文件。
“配置文件说明：
程序会自动生成 alist_versions.cfg 和 alist_current.cfg 文件
**一定不要删这些配置文件，否则您的版本信息会丢失！**
**配置文件与程序在同一目录，便于备份和迁移”**
